## 出来ること

## 写真の削除
いらなくなった写真を削除することが出来ます。
削除する際は、右上の削除ボタンから削除することが出来ます。
一度アラートが表示されるので、確認後削除するボタンを押してください。

## カメラ機能
カメラにて写真を撮ることができます。
カメラロールから右上のカメラボタンを押すことでカメラに飛びます。
